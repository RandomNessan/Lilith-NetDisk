一个简易的网盘系统


运行环境>
1. Nginx 1.24
2. PHP 7.4


部署流程>
1. 上传源码到根目录
2. 给/dl 和 /data 文件夹赋予766权限
3. 将nginx.txt中的文件内容粘贴入URL Rewrite中
4. 初始登录信息>>>账号: admin      ;密码：admin123
5. 将nginx_edit.txt中的内容粘贴在nginx配置中(点击"网站域名"===>"配置文件")

使用说明>
1. 可以在管理页面上传文件，网速过慢的情况下大于100MB的文件有可能上传失败
2. 上传失败的情况下可以直接把文件通过ftp或者panel放入 /dl 文件夹，然后同步即可获得链接


CDN配置>
# 没有配置CDN的情况下不用管这条
1. 源站协议: http; 源站地址: 服务器ip
2. 务必开启端口跟随，否则下载链接无法正常解析
3. 启用源站
